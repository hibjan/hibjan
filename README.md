## 🚀 Featured Projects

- **[Multimedia Collection Explorer](https://github.com/hibjan/TFG)** *(Bachelor's Thesis)* — Full-stack web app for navigating large multimedia datasets through rich metadata and relational links. Integrates public datasets (TMDb, DBLP). Containerized with Docker + Cloudflare tunnel for zero-port production deploy. &nbsp; `Java` `Tomcat` `PostgreSQL` `JavaScript` `Vite` `Docker`
- **[Tiny Language Compiler](https://github.com/hibjan/Procesadores-de-Lenguajes)** — Complete four-phase language processor for a prototypical imperative language: lexical analysis, parsing, abstract syntax tree construction, and semantic checking + code generation. &nbsp; `Java` `Lex` `Compilers`
- **[Concurrent P2P File-Sharing Server](https://github.com/hibjan/PC)** — Client/server peer-to-peer system in Java demonstrating concurrency primitives (locks, monitors), message-based protocols, user registry, shared library management, and P2P downloads with a GUI client. &nbsp; `Java` `Maven` `Concurrency` `Networking`
